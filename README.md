# GDP.v2

Plantilla fullstack generada con Node.js.

## Stack

- React + Vite + TypeScript
- Express + TypeScript
- Prisma + PostgreSQL

## Comandos

```bash
npm run dev
npm run frontend
npm run backend
npm run db:migrate
npm run db:generate
npm run db:validate
npm run db:studio
npm run db:up
npm run db:down
npm run build
```

## Primeros pasos

1. Copia `backend/.env.example` a `backend/.env`.
2. Ajusta `DATABASE_URL`.
3. Ejecuta `npm run db:up`.
4. Ejecuta `npm run db:migrate`.
5. Ejecuta `npm run dev`.
